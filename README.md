# Blaster Sound Hunt

Браузерна аркадна гра, де фізичний іграшковий бластер працює як контролер:

- камера відстежує помаранчеве дуло та синій корпус, визначає напрямок ствола й керує прицілом;
- мікрофон розпізнає різкий звук і активує постріл.

## Запуск

1. Відкрийте `index.html` у сучасному браузері або запустіть локальний сервер у цій папці.
2. Натисніть «Налаштувати камеру», дозвольте доступ і покажіть камері помаранчеве дуло разом із синім корпусом.
3. Зафіксуйте верхню ліву й нижню праву позиції для точного наведення.
4. Натисніть «Калібрувати бластер», дозвольте доступ до мікрофона і зробіть три постріли.
5. Камера рухає приціл, а звук бластера стріляє. Миша, палець, Space і клік залишаються резервним керуванням.

Для камери й мікрофона браузеру потрібен `localhost` або HTTPS:

```bash
python3 -m http.server 4173
```

Потім відкрийте `http://localhost:4173`.
